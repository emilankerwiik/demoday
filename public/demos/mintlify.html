Quickstart

Get your documentation up and running in under five minutes.

ⓘ Mintlify automatically deploys your docs from a GitHub repository. Push changes and your docs update in seconds.

Installation

Install the Mintlify CLI to preview changes locally before pushing.

Copy npm i -g mintlify

Project structure

Your docs live alongside your code. The mint.json file configures navigation, branding, and integrations.

Copy # mint.json { "name": "Acme Docs", "navigation": [ { "group": "Get Started", "pages": ["quickstart", "development"] } ], "colors": { "primary": "#3DD68C" } }

Local preview

Run the dev server to see changes in real time.

Copy mintlify dev # ✓ Running on http://localhost:3000
